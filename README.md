<h2 id="Project"> Projeto </h2>

O projeto foi desenvolvido na plataforma <a href="web.digitalinnovation.one/">Digital Innovation One</a> o desafio foi criar o jogo Gênesis utilizando apenas HTML, CSS e Javascript de forma introdutória, sendo o principal conceito, CSS Grid, manipulação de Array e Arrow Functions. Foi colocado em pratica tambem conhecimentos sobre logica de programação, condicionais e laços de repetição.
